# tunica-scripts

Scripts and utilities for processing data related to the Tunica Language Resources project. Intended for internal use by APS staff.

The code for the actual Jekyll site can be found [here](https://github.com/AmericanPhilosophicalSociety/tunica-language-resources).

## data

Data backups: FLEx data dump, FLEx XML exports, and full-notebook HTML files as of October 2025. Also includes documentation for how to export and back up FLEx data in various forms.

## escriptorium

Scripts and workflows related to transcribing and aligning Tunica notebooks on Escriptorium. These were used in the production of training data. Also includes plain-text transcripts of notebook 12 to be used in alignment.

## editioncrafter

Scripts and workflows related to the [Tunica Language Resources notebooks viewer](https://tunica-biloxi.amphilsoc.org/notebooks/). See create-jekyll-ec-files.md for documentation on how to run the full process, including exporting FLEx data to display in the notebooks  viewer.
